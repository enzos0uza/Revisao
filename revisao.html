HTML

HTML, que significa HyperText Markup Language, é a linguagem usada para estruturar o conteúdo de páginas da web. Com ela, é possível organizar textos, imagens, links, vídeos e outros elementos de forma que os navegadores entendam e exibam corretamente ao usuário. Um documento HTML começa com a declaração DOCTYPE html, que informa ao navegador que o conteúdo segue o padrão HTML5. Depois disso, toda a estrutura fica dentro da tag html, que contém duas seções principais: o head e o body. O head guarda informações que não aparecem diretamente na página, como o título que aparece na aba do navegador (title), a codificação de caracteres (meta) e links para arquivos externos como CSS. Já o body é onde vai o conteúdo visível da página, como textos, imagens e botões. Entre as tags mais importantes do HTML estão as de títulos (h1 até h6), que servem para destacar partes do conteúdo, e os parágrafos p, que organizam o texto. Também são muito comuns as tags de links a, imagens img e listas ul, ol, li. Para organizar blocos de conteúdo, usa-se a tag div, e para trechos menores dentro do texto, a tag span. Outro ponto importante é o uso do atributo type, que aparece em várias tags para definir o tipo de conteúdo ou comportamento. Por exemplo, na tag script, o atributo type especifica que tipo de linguagem está sendo usada (geralmente text/javascript, embora hoje isso já não seja obrigatório). Em formulários, o type é essencial na tag input para determinar o tipo de campo: text para texto comum, email para endereços de e-mail, checkbox para caixas de seleção, radio para botões de opção e submit para o botão de envio.


CSS

CSS, ou folha de estilo em cascata, é usado para definir a parte mais visual do site. As principais classificações do css são ID e as Classes, que por sua vezes são declaradas com "#" para ID's e com "." para classes, é importante de se ressaltar que, quando uma palavra é nativa do próprio html não é necessário declara-la com . ou # apenas escreval a palavra e o comando irá ser aplicado a tudo oque estiver dentro como por exemplo o boddy, que se colocado no css vai aplicar todas as funções dentro do mesmo. Algun comandos básicos e oque eles fazem: Display flex: transforma uma div em uma div flex , permitindo que os elementos filhos (os itens flex) sejam posicionados com muito mais controle. JC: center: centraliza os elementos. Color: usado para mudar a cor de um elemento; Font family: usada para mudar a fonte textual. BC color: usado para editar a cor do background


Java Script

Conecta ao html por meio do comando "script" no head, geralmente se usa camel Case, as variáveis podem ser definidas como let, que pode ser variável e const, que é uma constante, dentre essas duas a variável pode ser string, numeral, booleana (verdadeira ou falsa), array (vetor) ou um objeto.


Git

Git é um sistema de controle de versão usado para rastrear e gerenciar alterações em arquivos, principalmente em projetos de desenvolvimento de software. Ele permite que desenvolvedores salvem versões do código (commits), trabalhem em equipe com branches (ramificações) e revertam mudanças quando necessário. O fluxo básico envolve iniciar um repositório com `git init`, adicionar arquivos com `git add`, salvar alterações com `git commit`, e, se estiver usando um repositório online como o GitHub, sincronizar com `git push` e `git pull`. Com Git, é possível manter o código organizado, colaborar de forma eficiente e garantir um histórico claro das modificações ao longo do tempo.
